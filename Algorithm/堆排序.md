## 过程
1. 构造堆：将元素数组重新组织安排进一个[[二叉堆]]中。
2. 下沉排序：从堆中按递减顺序（取根结点）取出所有元素得到排序结果。

### 堆的构造
*方法 1*：
从左到右遍历数组^[遍历，至少是 N 次。]，用 [[二叉堆#^swim]]方法保证扫描指针左侧的所有元素已经是一个堆。
时间复杂度：NlogN

*方法 2*：
从右到左用[[二叉堆#^sink]]方法构造子堆。

## 代码实现
```java
public static void sort(Comparable[] a){
	int N = a.length;
	for (int k = N/2; k >= 1; k--){
		sink(a, k, N);
	}
	
	while (N > 1){
		exch(a, 1, N--);
		sink(a, 1, N);
	}
}

```
